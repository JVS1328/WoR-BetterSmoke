## How to Install:
- Download the file
- Open file explorer
- Go to: C:\Program Files (x86)\Steam\steamapps\common\War of Rights\Assets (or wherever this folder is for you)
- Drag the file into the Assets folder
- Launch War of Rights. Enjoy!
